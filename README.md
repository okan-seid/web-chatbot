# Web Chatbot for a news website

Дипломна Работа - Разработване на уеб чат бот за новинарски сайт - https://razgradnews.net<br/>
Live at - https://thriving-kleicha-0a562e.netlify.app (results shown may NOT be up to date due to need of manual web scraping)

![logo](https://github.com/okan-seid/web-chatbot/blob/5654f085911f06428e5eec2ffa227cc8d8fa6c68/img/logo.png)

## Използвани Технологии
- Python (web scraping)
- Node.js + Express (backend)
- MongoDB (database)
- HTML, CSS, JavaScript (frontend)
- MongoDB Atlas (cloud database)
- Render (cloud backend)
- Netlify (cloud frontend)
